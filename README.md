# vuln-HTTP-Desync-Scanner
Identifies HTTP desynchronization vulnerabilities by sending crafted HTTP requests with ambiguous Content-Length and Transfer-Encoding headers and analyzing server responses for discrepancies and potential request smuggling. - Focused on Assess vulnerabilities in web applications by performing scans and providing detailed reports

## Install
`git clone https://github.com/ShadowStrikeHQ/vuln-http-desync-scanner`

## Usage
`./vuln-http-desync-scanner [params]`

## Parameters
- `-h`: Show help message and exit
- `-t`: No description provided
- `-v`: Enable verbose output
